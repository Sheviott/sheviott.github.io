# Color Customizer

Небольшой инструмент для быстрой настройки цветовой схемы сайта — помогает подбирать и тестировать цвета элементов прямо в браузере. Создан для упрощения работы над цветовым дизайном личного сайта.

## Зачем это нужно
- Быстро менять цвета контента без правки CSS.
- Визуально подбирать сочетания и сразу видеть результат.
- Экспортировать текущую палитру.

## Как пользоваться
1. Откройте в браузере.
2. В панели управления выберите элемент и задайте цвет.
3. Изменения применяются мгновенно.

## Технологии
- HTML5
- CSS3
- React (react-colorful, react-dom, react-icons, react-redux, react-router, react-router-dom)
- TypeScript

## Установка
```npm run dev```  - запуск проекта

## Структура проекта
- `index.html` — разметка страницы.
- `style.css` — базовые стили интерфейса.

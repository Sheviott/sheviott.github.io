# Color Customizer
Мой личный проект над которым я активно работаю в данный момент.
Небольшой инструмент для быстрой настройки цветовой схемы сайта — помогает подбирать и тестировать цвета элементов прямо в браузере.

## Зачем это нужно
- Быстро менять цвета контента без правки CSS.
- Визуально подбирать сочетания и сразу видеть результат.
- Экспортировать текущую палитру.

## Как пользоваться
1. Откройте демо <https://sheviott.github.io/> в браузере.
2. В панели управления выберите элемент и задайте цвет.


## Технологии
- HTML5
- CSS3
- React (react-colorful, react-dom, react-icons, react-redux, react-router, react-router-dom)
- TypeScript

## Установка
```npm run dev```  - запуск проекта
